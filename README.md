# Workloads hub Agent

Mutli Agents and tools using Python, Streamlit and AutoGen.

To run it locally:

```bash
$ git clone <this-repo-url>

$ cd <this-repo-folder>

$ python -m venv venv

$ venv\Scripts\activate  # or source venv/bin/activate in Linux/Mac

$ pip3 install -r requirements.txt

$ streamlit run app.py
```
